# 概率场景树 (Scenario Tree)

**时间戳**：2026-04-14 | **基础事件**：海上封锁宣布（2026-04-13）

---

## 30天窗口（至2026-05-14）

### 节点A：伊朗对海上封锁的初步反应

**分支A1：伊朗宣布接受谈判条件（概率30-40%）**
- **触发条件**：Vance团队发布"有期限的解除条件"，沙特确认调停意愿
- **政治信号**：Mojtaba或IRGC官员发表"与美国对话的可能性"讲话
- **可观察指标**：伊朗向伊斯兰堡通知"重启谈判意愿"
- **结果路径**：→ 节点B（临时协议路径）

**分支A2：伊朗宣布对抗封锁，但不升级军事冲突（概率35-45%）**
- **触发条件**：Mojtaba宣布"长期抵抗战争"，伊朗宣布核冻结至70%作为"底线"
- **政治信号**：IRGC发表"防御性回应"声明，但不主动攻击美军
- **可观察指标**：伊朗向霍尔木兹海峡增派船队，但无攻击动作
- **结果路径**：→ 节点B（低烈度对抗路径）或 → 节点C（升级触发路径）

**分支A3：伊朗以代理人反击（概率15-25%）**
- **触发条件**：伊朗宣布"对美国盟国的防御反击权"，真主党/胡塞准备报复
- **政治信号**：Mojtaba发表"任何进一步的经济压力都会导致军事反制"讲话
- **可观察指标**：IRGC Quds Force指挥官发表"代理人准备就绪"声明；真主党、胡塞增加公开军事演练
- **结果路径**：→ 节点C（代理人反击路径）

## 30-90天窗口（至2026-07-14）

### 节点B：临时协议与长期冻结

**分支B1：两周内达成临时协议（概率25-35%）**
- **充要条件**：
  1. 美国同意解除部分海上封锁（允许伊朗石油出口50%容量）
  2. 伊朗同意核冻结至60%+，允许IAEA现场验证
  3. 两周内见效，建立60天"再谈判"窗口
- **可观察信号**：
  - Vance团队发布"框架协议"声明
  - 伊朗IAEA代表宣布"欢迎国际验证"
  - 美国暂停对伊朗新目标的轰炸
- **结果**：进入正式停火期 → 路径A（胜利停火）概率上升至50%+
- **时间戳**：约2026-04-21或2026-04-28

**分支B2：谈判失败，进入低烈度对抗（概率40-50%）**
- **触发条件**：
  1. Vance与伊朗代表在停火条件上分歧（美国坚持彻底解除核计划，伊朗拒绝）
  2. Trump政府内部对伊朗政策分歧（Vance vs Defense Secretary）
  3. 以色列压力美国"不应与伊朗妥协"
- **可观察信号**：
  - 谈判代表停止会面
  - 美国国防部发表强硬立场声明
  - 伊朗宣布"美国不诚心，继续自卫准备"
- **结果**：进入冻结期，双方都无力升级 → 路径B（结构性冻结）概率上升
- **时间戳**：约2026-05-14-07-14

**分支B3：临时协议后的长期谈判困局（概率20-30%）**
- **触发条件**：短期协议达成，但60天后谈判再度陷入僵局
- **可观察信号**：
  - 伊朗宣布"美国没有兑现承诺"
  - 美国指责伊朗"暗中浓缩"（尽管协议要求冻结）
  - 沙特与卡塔尔同时发表"失望"声明
- **结果**：从临时协议滑向长期冻结，或升级到核设施打击
- **时间戳**：约2026-07-14 后

## 90-180天窗口（至2026-10-14）

### 节点C：战争升级与内部政治

**分支C1：美国对核设施轰炸（概率20-30%）**
- **触发条件**：
  1. IAEA报告伊朗浓缩丰度升至75-80%
  2. 以色列向美国施压"必须打击Natanz"
  3. Trump政府决定"一劳永逸地解决伊朗核问题"
- **可观察先行信号**（7-14天提前）：
  - 美国国务卿发表"伊朗违反承诺"声明
  - B-2轰炸机从美国本土起飞，或在中东地区增加飞行强度
  - 美国国防部发表"核设施打击的军事计划已准备就绪"评论
- **军事行动内容**：
  - 精准轰炸Natanz（地表加浓缩设施），预期造成5-10年延迟
  - 可能附带轰炸Fordow（地下，难度更高），或保留以做未来威胁
- **伊朗反应**（24-72小时）：
  - IRGC最高指挥部发表"全面反击"宣言
  - 伊朗宣布退出NPT
  - 代理人（真主党、胡塞、PMU）全面发动对沙特、以色列、美军基地的打击
- **结果路径**：→ 升级至第8-9级（核设施打击+大规模代理人反击）
- **估计时间**：2026-07-14 至 2026-09-14 期间
- **概率区间**：20-30%

**分支C2：伊朗加速浓缩，但美国不轰炸（概率25-35%）**
- **触发条件**：
  1. IAEA报告伊朗浓缩丰度升至75-80%，但美国评估"尚未达到战争阈值"
  2. Trump政府内部对轰炸的成本-收益评估分歧
  3. 中国、俄罗斯向美国传递"任何轰炸都会导致石油禁运"的威胁
- **伊朗行为**：
  - 继续加速浓缩，但公开保证"不进行核试验"
  - 与中俄加强防御合作（防空系统、防空导弹部署）
  - 向国际社会宣布"核防御能力已足够，不再发展"
- **结果路径**：→ 长期冻结（路径B），伊朗成为事实上的核阈值国家
- **估计时间**：2026-07-14 至 2026-10-14
- **概率区间**：25-35%

**分支C3：Mojtaba权力面临内部挑战（概率10-20%）**
- **触发条件**：
  1. 海上封锁持续3个月，伊朗经济恶化（失业率>15%）
  2. IRGC内部对战争成本的抱怨增加
  3. 某IRGC派系对Mojtaba的权力提出公开质疑
- **可观察先行信号**（2-4周提前）：
  - IRGC最高指挥部召开特别会议（超过正常频率）
  - 某将领就战争政策发表与Mojtaba不同的言论
  - 伊朗议会多数派对继续战争表示保留意见
- **内部权力斗争过程**：
  - 可能形成"激进派"（要求升级）和"务实派"（要求和平）的公开分裂
  - Mojtaba可能失去部分IRGC派系的支持
  - 权力可能向"集体领导制"转变（Guardian Council + IRGC共同主导）
- **结果路径**：→ 权力重组，战争政策可能调整（向停火/冻结倾斜）
- **估计时间**：2026-06-14 至 2026-09-14
- **概率区间**：10-20%

## 关键分支点与触发指标

### 分支点1：是否在2026-04-21前达成谈判框架？

**触发指标**：
- Vance团队与伊朗代表的会面频率（每周1次 = 谈判进行中）
- 美国是否公布"解除封锁的条件清单"（公布 = 谈判诚意）
- 沙特与卡塔尔是否发表"支持谈判"声明（支持 = 调停可行）

**概率评估**：
- 若三指标全部满足 → A路径概率↑ 至50%+
- 若仅有1个指标满足 → A路径概率↓ 至20%
- 若全不满足 → B/C路径概率↑ 至70%+

---

### 分支点2：是否在2026-07-14前IAEA报告浓缩至75%+？

**触发指标**：
- IAEA月报中伊朗浓缩丰度数据（核心）
- 卫星图像显示Natanz/Fordow新离心机安装（证实数据）
- 伊朗官方宣布"浓缩加速"（政治信号）

**概率评估**：
- 若浓缩升至75%+ → C1路径（核设施打击）概率↑ 至40%+
- 若浓缩维持在60-70% → B/C2路径概率↑ 至60%+
- 若浓缩下降 → A路径（停火）概率↑ 至60%+

---

### 分支点3：IRGC内部是否出现明显分裂？

**触发指标**：
- IRGC最高指挥部将领之间是否发表对立观点（3个以上将领）
- IRGC高层人事调任频率（若超过1次/月 → 清洗信号）
- 某IRGC派系（如Basij）是否拒绝执行命令（一个以上地区）

**概率评估**：
- 若分裂明显 → C3路径（权力重组）概率↑ 至30%+
- 若无明显分裂 → Mojtaba权力稳定，A/B路径概率不变
- 若分裂严重（Quds Force vs Revolutionary Guard） → 权力可能瓦解，新的权力格局形成

## 三方并行独立变量

以下变量在分支树中是**并行独立**的，概率相加不等于100%：

1. **伊朗核浓缩速度**（快速vs缓慢）与**谈判进度**（快速vs缓慢）可以并行：
   - 例：可能同时浓缩加速+谈判进行 → 中等概率

2. **Mojtaba权力巩固进度**与**战争持续性**并行：
   - 例：权力快速巩固+冲突冻结 → 稳定冻结局面

3. **美国打击决心**与**伊朗防御准备**并行：
   - 例：美国有意打击+伊朗完成防御部署 → 打击成本上升
