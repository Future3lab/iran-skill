# 历史模式→当代映射矩阵
# Historical Pattern → Contemporary Mapping Matrix

**时间戳**：2026-04-14 | **功能**：将 2500 年波斯帝国经验转化为可操作的分析变量

---

## 使用方法

本矩阵的功能是**桥接**：把 `/history/leaders/` 中的心智模型
与 `/frameworks/` 和 `/decisionmakers/` 中的当代分析变量连接起来。

**调用时机**：
- 当框架分析给出「概率 X%」但无法解释「为什么是 X 而不是 Y」时
- 当 decisionmaker 模型的 `key_unknowns` 涉及文化/心理因素时
- 当需要向非伊朗专家解释伊朗行为的深层逻辑时

## 矩阵一：权力合法性模式的 2500 年循环

| 历史时期 | 合法性来源 | 当代对应 | Skill 变量 |
|----------|-----------|---------|-----------|
| 阿契美尼德 (550-330 BCE) | 军事征服 + 制度效率 + Farr（天命） | IRGC 的「保卫革命」叙事 = 军事合法性 | IRGC 行为模型 |
| 萨珊 (224-651) | 琐罗亚斯德国教 + 王权神授 | velayat-e faqih = 什叶派版王权神授 | Mojtaba 合法性 |
| 萨法维 (1501-1736) | 什叶派教派身份 + 帝国建设 | 伊斯兰共和国 = 萨法维模式 2.0 | domestic-fragility 全局 |
| 恺加 (1789-1925) | 衰退期——合法性持续流失 | 如果经济崩溃+军事失败持续→参照恺加末期 | 石油收入变量 |
| 巴列维 (1925-1979) | 现代化 + 古波斯复兴主义 | IRGC 试图同时使用伊斯兰和波斯民族两条线 | （新维度） |
| 伊斯兰共和国 (1979-) | 什叶派革命 + 反帝国主义 | 当前体制——但两条腿都在动摇 | 所有框架 |

**核心洞察**：伊朗政权的合法性从来不是单一来源的。
每个朝代都同时使用多条合法性线索，当一条断裂时切换到另一条。
Mojtaba 当前面临的挑战是：革命合法性在消退，
但他既没有 Khamenei Sr. 的宗教权威，也没有足够的军事胜利来弥补。

## 矩阵二：「主权创伤」的递归积累

| 事件 | 年份 | 创伤机制 | 当代触发物 | 反应强度 |
|------|------|---------|-----------|---------|
| 恺加石油特许权 | 1901 | 经济主权让渡→外国控制核心资源 | 任何「零浓缩」要求 | ★★★★★ |
| 英俄瓜分伊朗 | 1907 | 大国无视伊朗主权直接瓜分 | 任何「不经伊朗同意」的安排 | ★★★★☆ |
| 二战英苏占领 | 1941 | 中立国被强制占领 | 军事基地/领空使用谈判 | ★★★☆☆ |
| CIA/MI6 政变 | 1953 | 民选政府被外部推翻 | 任何美国的「政权更迭」暗示 | ★★★★★ |
| 美国撕毁 JCPOA | 2018 | 签字的国际协议被单方面撕毁 | 任何新协议的「持久性」疑问 | ★★★★☆ |
| Khamenei Sr. 空袭身亡 | 2026 | 最高领袖在外国军事行动中死亡 | 所有后续谈判的「信任基线」 | ★★★★★ |

**核心洞察**：这些创伤不是孤立事件——它们在集体记忆中递归积累。
每一次新的「主权侵犯」都会激活所有前次记忆。
2026 空袭不仅仅是一次军事行动——
它在伊朗集体心理中同时激活了 1907+1941+1953+2018 的全部创伤。
这就是为什么「施压让伊朗妥协」的逻辑在结构上是错误的。

**Skill 应用**：当评估伊朗在谈判中的「让步空间」时，
必须把这个递归创伤考虑在内——
每一个涉及「主权让渡」的条款，在伊朗决策者的心理中
都会被自动乘以一个「历史创伤系数」。

## 矩阵三：帝国更替周期与政权脆弱性

| 阶段 | 历史案例 | 典型特征 | 当前位置评估 |
|------|---------|---------|------------|
| **1. 创建期** | 居鲁士/阿巴斯一世/霍梅尼 | 魅力领袖 + 意识形态动力 + 军事胜利 | ✗ 已过 |
| **2. 巩固期** | 大流士一世/Khamenei Sr. | 制度建设 + 权力分配 + 精英妥协 | ✗ 已过 |
| **3. 传承期** | Mojtaba 继位 | 继承人合法性不足 + IRGC 拱卫 + 外部危机 | ✓ **当前位置** |
| **4. 衰退前期** | 纳赛尔丁·沙阿 / 萨法维后期 | 改革口号 > 实质 + 精英分裂开始 | 6-18个月后可能 |
| **5. 危机期** | 恺加末期 / 巴列维末期 | 多重压力叠加 + 安全部队忠诚度动摇 | 概率 <10%（当前） |
| **6. 更替期** | 1925 / 1979 | 新势力取代旧体制 | 当前不适用 |

**核心洞察**：伊朗当前处于「传承期」——
这是所有帝国周期中最脆弱的阶段之一。
Mojtaba 面临的结构性挑战和历史上所有「继承者」几乎相同：
- 他不是创建者，缺少创建期的魅力合法性
- 他不是制度建设者，缺少巩固期的制度化权威
- 他唯一的权力基础是 IRGC——但 IRGC 有自己的利益逻辑

**与 domestic-fragility 框架的对接**：
- 当前综合脆弱性 4.0/10 = 传承期的正常值
- 如果石油收入在 3 个月内枯竭 → 可能推入衰退前期（6-7/10）
- 如果 IRGC 内部分裂信号明确 → 可能直接跳入危机期

## 矩阵四：「强人-衰弱-更替」周期的时间规律

| 周期 | 强人 | 衰弱期 | 更替 | 时间跨度 |
|------|------|--------|------|---------|
| 阿契美尼德→亚历山大 | 居鲁士-大流士 | 薛西斯后的持续衰退 | 亚历山大征服 | ~220年 |
| 萨珊→阿拉伯 | 沙普尔一世 | 后期国王能力下降 | 阿拉伯征服 | ~400年 |
| 萨法维→阿夫沙尔 | 阿巴斯一世 | 后继者无能 | 纳迪尔夺权 | ~130年 |
| 恺加→巴列维 | 纳赛尔丁 | 英俄压力下持续衰退 | 礼萨政变 | ~100年 |
| 巴列维→革命 | 礼萨·沙阿 | 穆罕默德·礼萨的脆弱平衡 | 1979革命 | ~54年 |
| 伊斯兰共和国 | 霍梅尼→Khamenei | Mojtaba 传承期 | ？ | 47年（至今） |

**方法论警告**：此表不预测「伊斯兰共和国何时结束」。
它展示的是一个结构性模式：
每个帝国/政权都经历创建→巩固→传承→衰退→更替的周期，
但时间跨度差异巨大（54 年到 400 年不等）。
「政权更迭」是长期尾部事件，不应作为短期分析的默认框架——
这与 SKILL.md 的硬性约束第六条（❌ 大规模抗议=即将垮台）一致。

## 诚实边界

1. 本矩阵是**分析工具，不是预测引擎**——它提供「为什么」，不提供「何时」
2. 历史模式映射是**类比推理**——类比的价值在于启发，风险在于过度简化
3. 文化心理变量的**权重无法量化**——它与结构性变量的交互方式未知
4. 本矩阵主要基于**英文学术来源**——波斯语/阿拉伯语一手资料的补充仍是优先级
5. 不同社会阶层对历史模式的感知不同——城市中产 vs 农村 vs 巴扎商人 vs IRGC 军人
