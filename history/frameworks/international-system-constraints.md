## 约束一：北约缺席——美国首次「独自打仗」

### 事实基线

这是冷战以来美国第一次在没有任何北约盟友直接参战的情况下发动重大军事行动。

- 德国：明确表示冲突「与北约无关」——北约是领土防御联盟，不适用于选择性战争
- 英国：未参与打击本身，限制了美军使用迪戈加西亚基地，但后来松动为提供「防御性支持」
- 法国：从国际法角度批评军事行动，在联合国要求紧急讨论
- 欧洲多国（希腊、意大利、德国）拒绝参与霍尔木兹海军任务
- 波罗的海国家（立陶宛、爱沙尼亚）态度最温和，但也要求美方先解释「战略目标」

### 为什么北约不帮忙

不是因为「胆怯」——而是三个结构性原因叠加：

1. **Trump 先前已经损害了联盟信任**：格陵兰危机（2026年1月）+ 关税战 + 退出多边框架 → 欧洲不愿为一个反复威胁自己的盟友承担风险
2. **乌克兰优先**：欧洲的核心安全关切是俄罗斯，不是伊朗。伊朗战争从乌克兰防御中分流资源和注意力
3. **国际法立场**：NATO 不承担支持选择性战争的义务，且欧洲缺乏在敌对行动进行中以武力重新开放霍尔木兹海峡的现实手段

### 对 Skill 概率的影响

| 变量 | 影响方向 | 理由 |
|------|---------|------|
| 封锁持续性 | **↓ 降低** | 没有盟友海军配合，美国独自维持全球最繁忙航道的封锁成本极高 |
| B 路径概率 | **↑ 上升** | 没有制裁联盟 = 欧洲/亚洲企业可以继续与伊朗交易 = 封锁漏洞多 |
| A 路径时间窗 | **↑ 缩短** | 北约裂痕本身成为 Trump 的国内政治问题，增加快速收场的压力 |
| Trump 行为 | 双向 | 一方面更想快速结束以回头处理北约问题；另一方面愤怒可能导致升级 |

### 关键信号追踪

- Trump 是否真的推进退出 NATO 的讨论？（截至4/14：威胁但未行动）
- 英法是否推动联合国框架介入？
- 欧洲是否在战后（而非战中）同意参与霍尔木兹航行自由任务？

## 约束二：中国——「不死不活」的经济命脉提供者

### 事实基线

中国在伊朗战争中的位置是精心校准的：
- 参与了停火斡旋（3/31 巴基斯坦-中国「五点倡议」）
- 帮助伊朗重建防空系统
- 但没有提供任何进攻性军援
- 中国油轮在 Trump 宣布封锁后仍在通行（Trump 选择性执法）

### 中国的三角计算

中国在伊朗问题上的决策不是「支不支持伊朗」——而是三个变量的同时优化：

| 变量 | 中国利益 | 行动 |
|------|---------|------|
| **能源安全** | 伊朗是中国石油来源之一，霍尔木兹关闭直接威胁能源供应 | 推动停火，但不会为伊朗打仗 |
| **去美元化** | 伊朗战争加速全球对美元体系的质疑 | 乐见美国消耗自己，但不想被拖入 |
| **台海筹码** | 美国在中东消耗军事资源 = 台海压力减轻 | 希望冲突持续但不扩大到中国利益区 |

### 中国的「不死不活」策略

中国对伊朗的最优策略不是「伊朗赢」或「伊朗输」——而是**「伊朗不死」**：
- 提供足够的经济命脉让伊朗不崩溃（石油贸易、防空系统、外交支持）
- 但不提供足够的军事支持让伊朗能赢（没有战斗机、没有反舰导弹、没有卫星情报）
- 这样伊朗就会持续消耗美国的注意力和资源——中国的最优结果

### 对 Skill 概率的影响

| 变量 | 影响方向 | 理由 |
|------|---------|------|
| 伊朗政权存活 | **↑ 增强** | 中国经济命脉防止石油收入完全枯竭 |
| B 路径（冻结冲突）| **↑ 显著上升** | 中国的「不死不活」策略直接制造冻结冲突条件 |
| 封锁有效性 | **↓ 降低** | 中国油轮继续通行 = 封锁存在巨大漏洞 |
| C 路径（政权更迭）| **↓ 进一步降低** | 中国不会允许伊朗崩溃——那会让美国在中东的手「解放」出来 |

### 与 Xi Jinping YAML 的对接

现有 `xi_jinping.yaml` 的 bluff_rate（60-80%）需要用本约束重新解读：
中国的「战略伙伴」语言 vs 实际军援的落差**不是虚伪——是策略**。
中国精确地给了伊朗「不死」的量，这是最优策略，不是「不够」。

## 约束三：俄乌战争——伊朗战争的「吸氧管」

### 联动机制

伊朗战争和俄乌战争不是两个独立事件——它们通过至少四条管道联动：

1. **防空系统竞争**：乌克兰和伊朗同时需要防空系统 → 全球防空供给紧张 → 给一方就意味着另一方短缺
2. **伊朗无人机→俄罗斯**：伊朗战前向俄罗斯供应 Shahed 无人机。战后伊朗自己需要，供给减少 → 乌克兰间接受益
3. **注意力竞争**：美国决策层的注意力是有限的。伊朗战争分流了乌克兰的政治关注
4. **油价联动**：伊朗战争推高油价 → 俄罗斯石油收入增加 → 俄罗斯战争能力增强 → 乌克兰处境恶化

### 对 Skill 的影响

- 欧洲不参与伊朗战争的最深层原因之一：每分散到中东的资源，都是从俄罗斯前线抽走的
- 乌克兰的泽连斯基主动提供打击伊朗无人机的经验 → 这是乌克兰换取关注的策略
- Putin 的 YAML 中「机会主义：收益油价+分散美国」被本约束完全验证

## 约束四：制裁联盟的缺失——封锁的「结构性漏洞」

### 历史对比

| 制裁案例 | 联盟完整性 | 效果 |
|----------|-----------|------|
| 伊朗制裁 2012-2015（奥巴马期）| 美+EU+日韩+联合国安理会 | 有效——迫使伊朗进入 JCPOA |
| 伊朗制裁 2018-2021（Trump 一期）| 美国单边 + 次级制裁威胁 | 部分有效——但中国/土耳其/伊拉克持续绕制 |
| 伊朗封锁 2026（当前）| 美国+以色列，无盟友 | 漏洞极大——中国油轮通行，欧洲未参与执法 |

### 核心洞察

**2012 年制裁之所以有效，不是因为美国强——而是因为联盟完整。**
当 EU 银行系统切断伊朗的 SWIFT 接入时，伊朗的贸易渠道被全面封锁。
2026 年没有这个条件：
- 欧洲不配合 → 伊朗可以通过欧洲中间商绕制
- 中国不配合 → 伊朗最大的石油客户继续购买
- 印度/土耳其/伊拉克 → 历史上一直是制裁绕制的主要渠道

### 对 Skill 概率的影响

- 封锁有效性 = 取决于联盟完整性 × 执行力度
- 当前联盟完整性 ≈ 40-50%（仅美以）→ 封锁效果打五折
- 这意味着：石油收入不会「完全枯竭」→ domestic-fragility 的变量四（石油收入断裂）的触发时间延长
- B 路径概率进一步上升

## 约束五：巴基斯坦/土耳其/沙特——新的调停三角

### 意外的新角色

这场战争中最意外的结构性变化是调停格局：
- **巴基斯坦**成为核心调停方（主办伊斯兰堡谈判）
- **土耳其**试图弥合美伊分歧
- **沙特/埃及**参与幕后斡旋
- **联合国**作用有限
- **欧洲/EU**被排除在外

这意味着：战后秩序（如果有的话）将是「南方主导」而非「西方主导」的。
这对伊朗来说是一个结构性优势——巴基斯坦/土耳其/沙特都比美国更理解伊朗的红线。

## 诚实边界

1. 国际体系变量的影响是**间接的**——它改变的是约束条件，不是直接决策
2. 北约裂痕的深度和持久性尚不确定——可能在战后迅速修复
3. 中国的「不死不活」策略是推断——北京不会公开承认这一逻辑
4. 联盟完整性的量化（40-50%）是粗略估计，不是精确测量
5. 调停格局（巴基斯坦主导）是新现象——历史参照有限
