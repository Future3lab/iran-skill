## 模式一：「施压→屈服」幻觉（Coercion Illusion）

**历史案例**：
- 越南（1964-73）：逐步升级轰炸 → 预期河内屈服 → 河内没有屈服
- 伊拉克制裁（1990-2003）：12 年全面制裁 → 预期政权垮台 → 政权存活，平民受苦
- 朝鲜（1950-今）：70 年制裁+军事威胁 → 预期朝鲜放弃核武 → 朝鲜成为核国家
- 古巴（1960-今）：60+ 年禁运 → 预期卡斯特罗政权垮台 → 政权存活至今

**模式结构**：
```
美国假设：施加足够的痛苦 → 对方理性计算 → 让步
实际结果：对方把「承受痛苦但不屈服」定义为胜利 → 施压反而强化抵抗意志
```

**根本原因**：美国决策者的「理性」假设对方的「理性」和自己相同（mirror imaging）。
但对于伊朗这类政权，「理性」的计算单位不是经济损失，而是尊严/生存/合法性。

**当前伊朗映射**：
- 2026 海上封锁 → 预期伊朗回到谈判桌 → 但伊朗可能把「扛住封锁」定义为胜利
- 霍梅尼 1988「喝毒酒」证明：伊朗可以承受 8 年战争的极端痛苦后才做出最低限度让步
- **Skill 对接**：Takeyh「生存即胜利」框架 + khomeini.yaml「喝毒酒」模型
- **概率影响**：B 路径（冻结冲突）的概率应上调 → 因为美国制裁/封锁几乎从不主动结束

## 模式二：Lebovic 四阶段循环（Mission Creep Spiral）

**理论来源**：James Lebovic,《Planning to Fail: The US Wars in Vietnam, Iraq, and Afghanistan》

**四阶段**：
1. **军事投入**（Engagement）：以有限目标开始→「外科手术式打击」/「精准行动」
2. **使命蔓延**（Extension）：初始目标未达成 → 扩大行动范围 → 目标膨胀
3. **资源天花板**（Limitation）：国内政治约束 → 不愿再增投 → 但也不愿撤退
4. **延迟撤离**（Disengagement）：寻找「能宣布的胜利」→ 离开 → 对手填补真空

**历史印证**：
- 越南：顾问(1961)→地面部队(1965)→50万人(1968)→越南化(1969)→撤军(1973)→西贡陷落(1975)
- 阿富汗：特种部队(2001)→国家建设(2003)→增兵(2009)→撤军谈判(2020)→喀布尔陷落(2021)
- 伊拉克：震慑行动(2003)→占领(2003)→增兵(2007)→撤军(2011)→ISIS崛起(2014)

**当前伊朗映射**：
- 阶段一（已完成）：2026-02-28 空战 =「精准打击核设施」
- 阶段二（当前风险）：封锁(4/13)→ 如果封锁无效 → 扩大打击范围？→ 目标膨胀？
- 阶段三（预测）：Trump 的国内政治约束（Vance 的孤立主义 + 选举周期）→ 资源天花板
- 阶段四（最可能）：Trump 寻找「能宣布的胜利」→ 离开 → 类比 2018 朝鲜峰会
- **Skill 对接**：war-termination A 路径本质上就是 Lebovic 阶段四
- **概率影响**：A 路径的「Trump 宣布胜利停火」不是 Trump 的个人怪癖——它是美国 80 年对外战争的结构性终局模式

## 模式三：「朝鲜模式」冻结（Permanent Stalemate Template）

**历史案例**：朝鲜半岛（1953-今）

**模式结构**：
1. 热战 → 停火 → 无和平条约
2. 持续军事对峙 + 经济制裁
3. 对手发展核武 → 美国制裁升级 → 对手继续发展 → 冻结
4. 偶发的「峰会外交」→ 无实质改变 → 回到冻结
5. 美国学会「与之共存」→ 注意力转移到下一个危机

**当前伊朗映射**：
- 2026 空战 → 如果无法达成协议 → B 路径（冻结冲突）= 伊朗版朝鲜模式
- 伊朗已经是核阈值国家 → 如果浓缩升至 80-90% 但不核试 → C2 路径 = 事实核国家冻结
- Trump 2018 金正恩峰会 = 完美参照：宣布「historic deal」→ 实际无任何条款改变
- **Skill 对接**：scenario-tree 的 B 路径 + C2 路径
- **关键洞察**：美国开启的经济战争/军事对峙**几乎从不主动结束**——古巴(60年)、朝鲜(70年)、伊朗(45年+)。这意味着 B 路径一旦形成，可能持续数十年。

## 模式四：选举周期驱动的「October Surprise」

**历史案例**：
- 1968：约翰逊停止轰炸越南（大选前）
- 1972：基辛格宣布「和平近在咫尺」（尼克松连任前）
- 1980：伊朗人质危机悬而未决 → 卡特连任失败
- 2020：Trump 与塔利班签署多哈协议（大选年）

**模式结构**：
美国外交政策被选举周期强制节律化：
- 新总统前 18 个月 = 最大行动窗口（可以升级）
- 中期选举前 6 个月 = 开始寻找「胜利叙事」
- 大选年 = 必须有可展示的成果 OR 最小化风险

**当前伊朗映射**：
- 2026 中期选举（11月）= Trump 需要在 6-8 个月内有「成果」
- 这解释了为什么 A 路径（胜利停火）的时间窗口集中在 2026 年 Q2-Q3
- 如果 2026 年 Q3 前无法达成 → 进入「冻结期」直到 2028 大选年再次打开窗口
- **Skill 对接**：Trump 的 `known_triggers.deescalate`（民调下滑→快速宣布胜利）
- **概率影响**：30 天窗口的谈判概率(15%)可能偏低——选举压力在 Q2-Q3 会显著推高

## 模式五：「打完就走」的 Jacksonian 传统

**理论来源**：Walter Russell Mead,《Special Providence》(2001)

**Mead 四学派与 Trump 的位置**：

| 学派 | 代表人物 | 核心逻辑 | Trump 契合度 |
|------|---------|---------|------------|
| **Jacksonian** | Andrew Jackson | 不惹事但被惹了就打死你，然后回家 | **★★★★★** |
| Jeffersonian | Thomas Jefferson | 少管闲事，保护自己就好 | ★★★☆☆（Vance 更接近） |
| Hamiltonian | Alexander Hamilton | 商业利益驱动，维护国际秩序有利于贸易 | ★★☆☆☆ |
| Wilsonian | Woodrow Wilson | 推广民主和人权是美国使命 | ☆☆☆☆☆ |

**Jacksonian 的核心特征**：
1. 不相信「改造世界」——打完就走，不搞国家建设
2. 惩罚性远征（punitive expedition）> 长期占领
3. 荣誉/面子 > 战略利益计算
4. 海军力量投射 > 地面部队投入
5. 「过度反应后快速退出」是常见模式

**Trump 对 Jacksonian 的具体模仿**：
- 2025-2026 对伊空战 = Jacksonian 惩罚性远征
- 海上封锁 = Jackson 式海军力量展示
- 拒绝地面部队 = Jacksonian 的「打完就走」
- 「能宣布的胜利」= Jacksonian 的荣誉满足点
- **关键洞察**：Jacksonian 从 1930 年代以来一直边缘化，主要在美国或其公民受到威胁或攻击时才参战，也愿意接受旨在惩罚或迫使其他国家就范的惩罚性远征或外科手术式打击

**Trump 不是在模仿的人**：
- ❌ 不是小布什（没有国家建设野心）
- ❌ 不是奥巴马（没有多边主义信念）
- ❌ 不是 LBJ（没有逐步升级的耐心）
- ✅ 更像 Nixon（大国博弈 + 虚张声势 + 寻找退出）+ Jackson（惩罚后回家）

**当前伊朗映射**：
- Trump 对伊策略 = Jacksonian 惩罚远征 + Nixonian 退出外交
- 这意味着：**地面入侵概率极低**（<5% 与 skill 一致）——Jacksonian 从不搞长期占领
- 也意味着：**封锁可能半途而废**——当成本超过「荣誉收益」时，Jacksonian 会收手
- **Skill 对接**：Trump 的 `decision_style` + `known_bluffs` + war-termination A 路径

## 模式六：情报失败的反复模式

**历史案例**：
- 1961 猪湾：CIA 高估流亡古巴人的能力
- 1964 东京湾：基于错误情报升级越南战争
- 2003 伊拉克 WMD：情报被政治化 → 不存在的大规模杀伤性武器成为战争理由
- 2021 阿富汗：高估阿富汗政府军的抵抗能力

**反复出现的错误模式**：
1. **Mirror Imaging**：假设对方的「理性」和自己相同
2. **Groupthink**：决策者低估了敌人的实力和失败的可能性，没有讨论替代方案，没有考虑如果事情出错该怎么办，也没有认真对待来自国务卿鲍威尔的反对意见
3. **Drunkard's Search**：在方便的地方找答案，而不是在正确的地方
4. **沉没成本谬误**：「已经投入这么多，不能浪费」
5. **选择性使用情报**：挑选支持既定结论的情报，忽略矛盾证据

**当前伊朗映射**：
- Mirror Imaging = 假设伊朗的「理性」是经济最大化 → 实际是尊严/生存最大化
- Groupthink 风险 = Trump 团队内部「Iran hawks」可能压制反对声音
  - 但 Vance 的存在是一个制衡 → 降低 groupthink 风险
- 情报政治化风险 = 如果 Trump 需要升级的理由 → 可能选择性使用浓缩数据
- **Skill 对接**：bias-check 矩阵 + Albright/Lewis 的核数据独立验证

## 对 Skill 的操作性建议

1. **评估封锁持续时间时**：参照古巴/朝鲜模式——美国制裁一旦开始几乎从不结束。B 路径概率应上调。

2. **评估 Trump 退出时间时**：参照选举周期模式——2026 Q2-Q3 是关键窗口。A 路径的时间预测可以更精确。

3. **评估地面入侵概率时**：参照 Jacksonian 传统——Trump 的 DNA 是「打完就走」，地面入侵概率应保持 <5%。

4. **评估「施压→屈服」预期时**：参照越南/伊拉克/朝鲜——历史成功率极低。这一信息应在分析中显式呈现。

5. **评估情报质量时**：参照伊拉克 WMD 教训——当政治需要与情报结论矛盾时，情报会被政治化。核数据需要独立验证源（Albright/Lewis）。

## 诚实边界

1. 历史类比永远是不完美的——2026 伊朗 ≠ 1965 越南 ≠ 2003 伊拉克
2. Trump 是一个高度非传统的总统——历史参照系可能低估他的不可预测性
3. 本文件聚焦美国侧的系统性偏差——不意味着伊朗侧没有同等严重的偏差
4. Mead 四学派框架是分析工具，不是精确分类系统——真实决策永远是混合型
