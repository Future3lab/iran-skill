# 预测用户工具包
# Forecasting Toolkit for Iran.skill

**时间戳**：2026-04-14 | **功能**：让用户从「有分析框架」到「能做决策」

---

## 一、问题入口模板（不知道该问什么时，从这里开始）

### 按角色选择

**如果你是投资者/交易员：**
1. 「未来 48 小时内，霍尔木兹海峡的通行状态会改变吗？这对 Brent 原油意味着什么？」
2. 「封锁在 30 天内被解除的概率是多少？我是否应该做多/做空中东能源？」
3. 「如果战争进入冻结冲突（B 路径），哪些行业/国家会受益，哪些会受损？」
4. 「伊朗里亚尔在接下来 90 天内的走向？制裁绕制的规模有多大？」

**如果你是政策制定者/外交官：**
1. 「在 4/21 停火到期前，我们有多大的窗口推动第二轮谈判？需要什么条件？」
2. 「如果我们提出 X 条件，伊朗会接受吗？他们的真实红线在哪？」
3. 「Netanyahu 在黎巴嫩的行动如何影响美伊谈判？我们如何管理三角博弈？」
4. 「从波斯历史来看，什么样的协议措辞伊朗能接受？」

**如果你是记者/编辑：**
1. 「今天的故事角度是什么？所有人都在报道的 vs 没人在报道但很重要的？」
2. 「用不同分析师的视角，同一事件有几种完全不同的解读？」
3. 「下一个会打破现有叙事的'意外事件'最可能是什么？」

**如果你是研究者/学生：**
1. 「用帝国周期框架，当前伊朗处于什么位置？历史上类似阶段的政权持续了多久？」
2. 「美国对伊朗的施压策略，在历史上哪些成功了哪些失败了？失败的共同原因是什么？」
3. 「Shahnameh 文化心理学如何解释伊朗在谈判中的行为？」

**通用万能问题（任何角色都可以用）：**
- 「现在最重要的三个变量是什么？分别往哪个方向走？」
- 「我最应该担心的'意外事件'是什么？」
- 「当前所有人的共识判断是什么？这个共识最可能在哪里出错？」

## 二、分层时间框架（不同时间尺度，不同变量）

### 48 小时窗口（战术层）

**核心问题**：今天到后天，什么会变？
**关键变量**：
- 封锁执行中是否有拦截事件
- Trump/Vance 的最新推文/声明
- 以色列对黎巴嫩的攻击强度
- 任何一方的军事异动

**概率表达**：用「可能性」而非百分比（「很可能/可能/不太可能」）
**信息衰减**：半衰期 6 小时——今天的判断明天就需要更新

### 7 天窗口（操作层）★当前最重要

**核心问题**：停火 4/21 到期，会发生什么？
**关键变量**：
- 第二轮谈判是否确定日期
- 巴基斯坦/土耳其斡旋进展
- 油价走势（接近 $100→$110？）
- 以色列-黎巴嫩华盛顿谈判（4/15）结果
- Mojtaba 是否有新公开表态

**概率表达**：用中心估计+区间（「55% [45-65%]」）
**信息衰减**：半衰期 24-48 小时

### 30 天窗口（战略层）

**核心问题**：封锁之后的下一阶段是什么？
**关键变量**：
- 浓缩谈判是否有实质进展（5年 vs 20 年暂停的着陆区）
- 封锁的实际经济效果（中国油轮是否被拦截）
- IRGC 薪资是否出现问题
- 社会层（抗议）是否回归

### 90 天窗口（结构层）

**核心问题**：这场战争走向哪条路径？
**关键变量**：
- A 路径（胜利停火）vs B 路径（冻结冲突）vs C 路径（升级）
- Mojtaba 权力巩固进度
- 选举周期压力（2026 中期选举倒计时）
- 中国-伊朗经济关系的深化程度

### 180 天窗口（历史层）

**核心问题**：结局是什么？
**关键变量**：
- 帝国周期阶段转换（阶段三→阶段四？）
- 朝鲜冻结模式是否形成
- 美伊关系的结构性定型

## 三、验证仪表盘（你赌了一个场景，盯着什么来验证）

### 如果你赌「A 路径——停火协议」

| 盯着什么 | 支持信号（你对了） | 反驳信号（你错了） |
|----------|------------------|------------------|
| 谈判进度 | 第二轮日期确定；双方措辞软化 | 双方恢复强硬声明；调停方退出 |
| Trump 推文 | 减少对伊攻击性言论；开始说「deal」 | 恢复「毁灭」威胁；开始怪罪调停方 |
| 油价 | 稳定或回落→压力减轻 | 突破 $110→全球压力增大但可能反向迫使谈判 |
| 以色列-黎巴嫩 | 4/15 谈判有实质→三角解开 | 以色列大规模升级黎巴嫩→伊朗退出谈判 |
| 霍尔木兹 | 伊朗恢复部分通行（善意信号） | IRGC 加强布雷/拦截 |
| **止损线** | **如果 4/19 前无第二轮谈判日期 + Trump 恢复「毁灭」措辞 → 放弃 A 路径赌注** |

### 如果你赌「B 路径——冻结冲突」

| 盯着什么 | 支持信号 | 反驳信号 |
|----------|---------|---------|
| 停火到期后 | 双方都不恢复大规模打击 | 任一方重启轰炸 |
| 封锁执行 | 选择性执法（中国油轮放行） | 全面执法（拦截所有船只） |
| 国际反应 | 欧洲/日韩开始适应新常态 | 40+国联合施压要求解决 |
| 中国角色 | 继续「不死不活」供给 | 突然加大军援或完全切断 |
| **止损线** | **如果双方在停火到期后 72 小时内恢复大规模交火 → B 路径失效** |

### 如果你赌「C 路径——升级」

| 盯着什么 | 支持信号 | 反驳信号 |
|----------|---------|---------|
| IAEA 报告 | 浓缩升至 75-80% | 浓缩冻结或下降 |
| 军事动态 | B-2 起飞/航母群集结 | 美军减少中东部署 |
| Trump 措辞 | 从「deal」回到「obliterate」 | 继续「they want to deal badly」 |
| Netanyahu | 公开推动美国「finish the job」 | 转向黎巴嫩谈判 |
| **止损线** | **如果第二轮谈判成功启动 + 伊朗恢复部分海峡通行 → 放弃 C 路径赌注** |

## 四、「如果-那么」因果链图（10 个最关键的触发事件）

### 触发 1：油价突破 $120

```
油价>$120 →
├── 全球经济衰退压力↑ → 40+国施压要求停火 → A路径概率↑
├── 美国国内通胀↑ → Trump政治压力↑ → 加速寻找退出
├── 伊朗实际收入↓（封锁下出口量低）但名义价格↑
└── 中国能源成本↑ → 中国调停动力↑
净效应：A路径↑↑，B路径↓
```

### 触发 2：中国油轮被美军拦截

```
中国油轮被拦 →
├── 中美关系急剧恶化 → 伊朗问题变成中美问题
├── 中国可能减少对伊支持（避免与美直接冲突）
│   └── 伊朗经济命脉真正断裂 → domestic-fragility↑↑
├── 或中国升级回应 → 全球贸易战升级
└── 所有场景概率需要完全重新计算
净效应：黑天鹅——当前框架可能失效
```

### 触发 3：Netanyahu 对黎巴嫩发动大规模地面进攻

```
以色列地面入侵深入黎巴嫩 →
├── 伊朗宣布停火无效 → 恢复霍尔木兹完全封锁
├── 真主党全面反击 → 以色列北部受攻
├── Trump 被迫在以色列和伊朗之间选边
│   ├── 选以色列 → 美伊谈判彻底破裂 → C路径↑↑
│   └── 压制以色列 → Netanyahu政治危机 → 可能是三角解开的转机
└── 黎巴嫩人道危机 → 欧洲被迫介入
净效应：A路径↓↓，C路径↑↑，或反转（Trump压制Netanyahu→解开三角）
```

### 触发 4：Mojtaba 首次公开重大演讲

```
Mojtaba 公开演讲 →
├── 如果措辞强硬（「永不屈服」）→ 谈判空间↓，B/C路径↑
├── 如果措辞务实（「和平与尊严」）→ 谈判窗口↑，A路径↑
├── 如果提及浓缩具体数字 → 他在划定个人红线
│   └── 与 Araghchi/Ghalibaf 的措辞对比 → 内部是否统一
└── 任何措辞都是极稀缺数据 → 全面更新 Mojtaba 行为模型
净效应：取决于内容——但无论如何都是高价值情报
```

### 触发 5：IRGC 高层人事重大变动

```
IRGC 将领被替换/调任 →
├── 如果是 Mojtaba 安插自己人 → 权力巩固信号 → 短期不稳+长期稳定
├── 如果是 IRGC 内部派系博弈 → 分裂信号 → domestic-fragility↑
├── 如果被替换者公开反对 → 罕见且极危险 → C3路径↑↑
└── 如果伴随战争政策调整 → 可能预示谈判立场变化
净效应：取决于方向——但任何IRGC人事变动都是P0追踪事件
```

### 触发 6：伊朗恢复部分霍尔木兹通行

```
伊朗允许非美盟友船只通行 →
├── 释放善意信号 → 谈判窗口↑ → A路径↑
├── 但保留对美/以船只封锁 → 维持筹码
├── 全球油价↓ → 经济压力缓解
├── Trump 可以宣布「伊朗在屈服」→ 加速A路径
└── 但 IRGC 失去通行费收入 → 内部有反对
净效应：A路径↑↑——这是最清晰的积极信号
```

### 触发 7：大规模伊朗国内抗议回归

```
抗议回归（>100城市） →
├── 如果在停火期间 → 政权面临内外双重压力
├── IRGC 必须分散力量（对外防御+对内镇压）
├── 库尔德/俾路支地区可能趁机升级
├── Mojtaba 被迫在「对外妥协」和「对内镇压」之间选择
│   └── 如果选择妥协→A路径↑ / 如果选择镇压→国际孤立↑
└── 社会层脉搏的所有指标需要紧急更新
净效应：domestic-fragility↑↑，A路径或C3路径↑（取决于IRGC反应）
```

### 触发 8：Trump 公开宣布「我们已经赢了，不需要协议」

```
Trump 宣布单方面「胜利」→
├── 封锁可能维持但不升级 → 进入B路径
├── 美国国内政治：「战争结束了」叙事 → 中期选举加分
├── 伊朗：「美国退缩了」叙事 → Mojtaba 可以宣布胜利
├── 实际上什么都没解决 → 核问题/代理人/海峡全部悬而未决
└── 类比 2018 朝鲜峰会后 → 宣布胜利但问题原封不动
净效应：B路径↑↑——最典型的Jacksonian「打完就走」
```

### 触发 9：IAEA 报告浓缩升至 80%+

```
浓缩接近武器级 →
├── 美国鹰派（Rubio）推动重启打击 → C1路径↑↑
├── 以色列单独行动的概率↑
├── 国际社会恐慌 → 但欧洲仍不愿参战
├── 伊朗可能同时释放「不核试」信号 → C2路径（核阈值国家冻结）
└── 中俄态度可能转变 → 安理会紧急会议
净效应：C路径↑↑，A路径↓↓
```

### 触发 10：4/21 停火到期无续约但双方均不开火

```
静默过期 →
├── 法律上停火结束，实际上无战斗 → 灰色状态
├── 封锁继续 / 霍尔木兹继续受控 → 低烈度对峙
├── 调停方持续穿梭 → 可能在数周后形成新的框架
├── 朝鲜冻结模式的起点 → B路径正式开始
└── 市场反应：如果无战斗 → 油价可能稳定在当前水平
净效应：B路径↑↑——这是「世界学会与之共存」的开始
```

## 五、场景-行动映射（基于预测，你该做什么）

### 投资者/交易员

| 场景 | 概率 | 行动建议 | 对冲 |
|------|------|---------|------|
| A路径（停火协议） | 25% | 如果有明确信号→逐步做空能源/做多新兴市场 | 保留 20% 能源多头作为对冲 |
| B路径（冻结冲突） | 40% | 油价将在 $90-110 区间震荡→区间交易策略 | 关注中国能源替代供应链 |
| C路径（升级） | 15% | 不下注——波动太大 | 持有黄金/美债作为尾部风险对冲 |
| 静默过期 | 20% | 最类似 B 路径——但短期波动更小 | 同 B 路径 |

### 政策制定者/外交官

| 场景 | 行动建议 |
|------|---------|
| A路径窗口打开 | 立即准备核验证机制草案；提前与 IAEA 协调；准备「双方都能宣布胜利」的措辞方案 |
| B路径形成 | 建立长期制裁管理机制；与中国讨论伊朗经济命脉的「可接受范围」；启动黎巴嫩独立谈判 |

### 记者/编辑

| 场景 | 故事角度 |
|------|---------|

| C路径 | 「升级的逻辑」→ 不是「疯狂」而是三方博弈的结构性产物 |
| 通用 | 「所有人都在看错地方」→ 盯着贝鲁特而非霍尔木兹 |

### 企业/供应链管理

| 场景 | 行动建议 |
|------|---------|
| 任何场景 | 建立绕道好望角的备选航线；评估中东供应商的替代方案；储备 3-6 个月关键原材料 |
| B路径 | 接受「长期中断」作为新常态→重构供应链 |

## 使用说明

1. **新用户**：从「问题入口模板」（第一节）开始，选择你的角色，挑一个问题问 Claude
2. **已有判断的用户**：直接跳到「验证仪表盘」（第三节），设定你的追踪指标和止损线
3. **需要做决策的用户**：看「场景-行动映射」（第五节），把概率翻译成行动
4. **需要深度分析的用户**：用「如果-那么」因果链（第四节），输入你关心的触发事件

**每个预测都应该附带三样东西**：
1. 概率判断（含审计链）
2. 验证仪表盘（盯什么、什么时候看、什么信号说明你错了）
3. 止损线（什么条件下放弃这个判断）

没有止损线的预测不是预测——是信仰。
